# PROJETO ORÇABEM

Projeto para a cadeira de Análise e Projeto de Sistemas (APS)

Gerenciar as finanças pessoais é essencial para alcançar estabilidade financeira e evitar dívidas desnecessárias. Um sistema de controle de gastos eficaz não apenas registra despesas, mas também fornece insights valiosos sobre padrões de gastos e áreas onde é possível economizar. Vamos explorar como um sistema completo de controle de gastos pode ser implementado, incluindo gráficos e planilhas para destacar desperdícios e controlar possíveis novas dívidas.
